# Redigram Bot

> Take top photos from Reddit and post them on Instagram

```
Usage of ./redigram:
  -dry
        Don't actually post the image
  -minscore int
        Minimum score (default 100)
  -password string
        Instagram Password
  -store string
        Storage directory (default "used")
  -sub string
        The Subreddit to pull from (default "memes")
  -username string
        Instagram Username
```
